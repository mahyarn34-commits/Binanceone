# ربات تلگرامی اسکن بازار کریپتو (Binance) + استراتژی چندتایم‌فریمی

این ربات به‌صورت خودکار:
1. جفت‌ارزهای پرحجم USDT در Binance رو اسکن می‌کنه (چون اسکن *همه‌ی* هزاران نماد در هر چرخه عملی نیست و به Rate Limit می‌خوره؛ روی حجم بالا تمرکز می‌کنه که همون کوین‌های مهم بازارن).
2. روند هر نماد رو در تایم‌فریم **1H** و **4H** با EMA9/21 + RSI14 + MACD می‌سنجه.
3. فقط برای نمادهایی که روند 1H و 4H **هم‌جهت** هستن، وارد مرحله‌ی بعد می‌شه.
4. روی تایم‌فریم **M15** استراتژی ورود رو با EMA9/21 + RSI14 + MACD + Bollinger Bands + VWAP + SuperTrend اجرا می‌کنه و در صورت تکمیل همه‌ی شرایط، سیگنال BUY/SELL با Entry/Stop Loss/Take Profit می‌سازه.
5. نتیجه رو به تلگرام می‌فرسته — هم به‌صورت خودکار (هر ۱۵ دقیقه، قابل تنظیم) و هم با دستور `/scan`.

⚠️ **این ابزار صرفاً یک دستیار تحلیل خودکار است، نه توصیه‌ی مالی.** هیچ سیستمی ۱۰۰٪ دقیق نیست؛ مدیریت ریسک و تصمیم نهایی همیشه با خودتونه.

## پیش‌نیازها

- یک اکانت تلگرام
- یک اکانت [Railway](https://railway.app) (رایگان برای شروع کافیه)
- **نیازی به API Key صرافی نیست** — دیتای قیمت از endpoint های عمومی Binance گرفته می‌شه.

---

## مرحله ۱: ساخت ربات تلگرام با BotFather

1. توی تلگرام دنبال `@BotFather` بگرد و باهاش چت کن.
2. دستور `/newbot` رو بفرست.
3. یک اسم برای ربات انتخاب کن (مثلاً `Crypto Signal Bot`).
4. یک یوزرنیم انتخاب کن که باید به `bot` ختم بشه (مثلاً `my_crypto_signal_bot`).
5. BotFather یک **توکن** بهت می‌ده، شبیه این:
   `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
   این توکن رو نگه دار — همون `TELEGRAM_BOT_TOKEN` هست.

---

## مرحله ۲: دیپلوی روی Railway

1. کل پوشه‌ی این پروژه (`crypto_signal_bot/`) رو در یک ریپازیتوری GitHub آپلود کن (یا مستقیم با Railway CLI دیپلوی کن).
2. توی Railway:
   - **New Project → Deploy from GitHub repo** رو بزن و ریپازیتوری رو انتخاب کن.
   - Railway به‌طور خودکار از `railway.json` / `Procfile` می‌فهمه که باید `python bot.py` رو اجرا کنه.
3. برو به تب **Variables** پروژه و این متغیرها رو اضافه کن:

   | Key | Value |
   |---|---|
   | `TELEGRAM_BOT_TOKEN` | توکنی که از BotFather گرفتی |
   | `TELEGRAM_CHAT_ID` | فعلاً خالی بذار (مرحله بعد پرش می‌کنیم) |
   | `TOP_N_SYMBOLS` | `40` (اختیاری) |
   | `SCAN_INTERVAL_MINUTES` | `15` (اختیاری) |

4. Deploy کن. اگه لاگ‌ها رو نگاه کنی باید ببینی: `Bot starting (polling mode)...`

---

## مرحله ۳: گرفتن Chat ID و فعال‌سازی گزارش خودکار

1. توی تلگرام برو سراغ ربات خودت و بزن `/start`.
2. ربات یک پیام برمی‌گردونه که توش `chat_id` نوشته شده.
3. اون عدد رو کپی کن، برگرد توی Railway → Variables → مقدار `TELEGRAM_CHAT_ID` رو بذار برابر همون عدد.
4. سرویس رو Restart کن (Railway خودش بعد از تغییر Variable یک ری‌دیپلوی می‌زنه).
5. از این به بعد، هر `SCAN_INTERVAL_MINUTES` دقیقه یک‌بار گزارش خودکار برات میاد.

## تنظیمات قابل شخصی‌سازی (متغیرهای محیطی)

| متغیر | پیش‌فرض | توضیح |
|---|---|---|
| `TOP_N_SYMBOLS` | 40 | تعداد پرحجم‌ترین نمادهای USDT برای اسکن |
| `MIN_QUOTE_VOLUME` | 5,000,000 | حداقل حجم ۲۴ساعته (دلار) برای ورود به اسکن |
| `SCAN_INTERVAL_MINUTES` | 15 | فاصله‌ی اسکن خودکار |
| `REQUEST_DELAY_SECONDS` | 0.15 | تأخیر بین درخواست‌ها به Binance (برای رعایت Rate Limit) |

## منطق استراتژی (خلاصه)

**۱H و ۴H (فیلتر روند):**
هر تایم‌فریم امتیازی از -۴ تا +۴ می‌گیره بر اساس: جهت EMA9/21، موقعیت قیمت نسبت به EMA21، RSI بالا یا پایین ۵۰، و علامت هیستوگرام MACD. اگه امتیاز ≥ ۲ → صعودی، ≤ -۲ → نزولی. فقط وقتی 1H و 4H **هر دو** هم‌جهت باشن، نماد وارد مرحله M15 می‌شه.

**M15 (سیگنال ورود):**
برای BUY باید هم‌زمان: SuperTrend صعودی، قیمت بالای VWAP، EMA9 بالای EMA21، RSI بین ۴۰-۶۵، MACD هیستوگرام مثبت. برای SELL برعکس. Stop Loss بر مبنای EMA21 ± نیم ATR و Take Profit با نسبت ریسک‌به‌ریوارد تقریبی ۱:۲ محاسبه می‌شه.

## محدودیت‌ها و نکات مهم (حتماً بخون)

- این کد از خود کلود (Claude) اجرای پیوسته نداره؛ باید روی Railway (یا هر هاست دیگه) دیپلویش کنی — که همین کار رو الان انجام دادیم.
- اسکن *کل* بازار (هزاران نماد) در هر چرخه عملی نیست؛ به همین خاطر روی N نماد پرحجم تمرکز شده. اگه خواستی، `TOP_N_SYMBOLS` رو زیاد کن ولی مراقب Rate Limit صرافی باش.
- SuperTrend و VWAP اینجا با فرمول استاندارد پیاده‌سازی شدن ولی می‌تونن با نسخه‌ی دقیق TradingView کمی تفاوت جزئی داشته باشن (تفاوت‌های گرد کردن/warm-up).
- این ربات **معامله نمی‌کنه**، فقط تحلیل و سیگنال می‌ده. اجرای معامله، حجم پوزیشن و مدیریت ریسک کاملاً بر عهده‌ی خودتونه.
- قبل از استفاده‌ی واقعی، حتماً چند روز نتایج رو مانیتور کن و با تحلیل دستی مقایسه کن.
